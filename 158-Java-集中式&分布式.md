# 158-Java-集中式&分布式.md

+  分布式应用（distributed application ）
    + 由不同的运行于分离的运行环境下的组件构成的应用程序，通常是在不同的平台上通过网络互联起来 
    + 建立在网络之上的软件系统

+ 典型的分布式应用
    + 双端式（客户机－服务器 Client/Server）
    + 三端式（客户机－中间件－服务器 Client/Middleware/Server） 
    + 多端式（客户机－多组件－多服务器 Client/Multiple Middleware/Multiple server）
